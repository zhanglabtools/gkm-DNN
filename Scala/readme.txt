Scala code for counting gapped k-mer and perform DNN training.

required software:
CUDA 8.0 (if using GPU)
nd4j-cuda-8.0-platform 0.8.0
deeplearning4j-core 0.8.0
datavec-api 0.8.0
arbiter-core 0.8.0

The Count package contains several Scala object for counting lmer-fv and gkm-fv.
The HelpFunc package contains configuration for several DNN models.
